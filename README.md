# Vowel Counter

A simple Python program that counts the number of vowels (`A`, `E`, `I`, `O`, `U`) in a text entered by the user.

## Features

- Takes a text input from the user.
- Counts both uppercase and lowercase vowels.
- Displays the total number of each vowel separately.

## How It Works

The program:

1. Asks the user to enter a text.
2. Loops through each character in the text.
3. Checks whether the character is a vowel.
4. Counts each vowel individually.
5. Prints the results.

## Example

```text
Write a text: Hello World

Aa:0 Ii:0 Oo:2 Ee:1 Uu:0
```

```text
Write a text: Artificial Intelligence

Aa:2 Ii:5 Oo:0 Ee:3 Uu:0
```

## Requirements

- Python 3.x

